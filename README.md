# calculaOMeuSalarioMD
Ferramenta de cálculo de salário a recibos verdes para médicos dentistas


Esta ferramenta calcula o salário líquido aproximado de um prestador de serviços contratado a recibos verdes. Ao valor bruto é descontado o valor estimado de pagamento à segurança social, considerando que és solteiro, e a retenção na fonte caso se justifique (obrigatória quando o rendimento anual ultrapassa os 10.000€) - obtém-se assim o salário líquido.

Também é feita uma comparação com o salário  mínimo nacional líquido, tendo em conta os subsídios de alimentação,  natal e férias.
